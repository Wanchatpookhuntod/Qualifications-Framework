👑 ภาพรวม Admin

Admin คือผู้ดูแลระบบที่มีสิทธิ์สูงสุด ทำหน้าที่ตั้งค่าโครงสร้างข้อมูลพื้นฐานให้ระบบพร้อมใช้งาน ก่อนที่บทบาทอื่นจะเริ่มดำเนินการได้

หน้าที่รายละเอียด
จัดการผู้ใช้สร้าง/แก้ไข/ลบบัญชีผู้ใช้ทุกบทบาท
จัดการคณะสร้างคณะวิชาและภาควิชา/สาขาวิชา
จัดการหลักสูตรสร้างหลักสูตรและนำเข้ารายวิชาจากไฟล์
จัดการรายวิชาเพิ่ม/ลบรายวิชาในแต่ละหลักสูตร
💡 Admin ควรตั้งค่าโครงสร้างข้อมูล (คณะ → หลักสูตร → รายวิชา → ผู้ใช้) ก่อนที่ Academic จะสร้างภาคการศึกษา

🔐 การเข้าสู่ระบบ

  1. เปิดเบราว์เซอร์แล้วไปที่ URL ของระบบ
  2. กรอก ชื่อผู้ใช้ และ รหัสผ่าน แล้วกด "เข้าสู่ระบบ"
  3. ระบบนำไปยัง Admin Dashboard โดยอัตโนมัติ
💡 ผู้ใช้ที่มีหลายบทบาทสามารถสลับบทบาทได้ โดยคลิกรูปโปรไฟล์ที่มุมขวาบน แล้วเลือก "สลับบทบาท"

👤 จัดการผู้ใช้งาน

เข้าเมนู Admin → จัดการผู้ใช้งาน

➕ เพิ่มผู้ใช้ใหม่

  1. กรอก ชื่อผู้ใช้ (Username) — ใช้สำหรับล็อกอิน ไม่สามารถซ้ำกันได้
  2. กรอก รหัสผ่าน — ระบบเข้ารหัสอัตโนมัติ
  3. กรอก ชื่อ-นามสกุล
  4. ติ๊กเลือก บทบาท ที่ต้องการ (เลือกได้หลายบทบาทพร้อมกัน):
    • instructor — อาจารย์ผู้สอน
    • head — หัวหน้าสาขา
    • academic — ฝ่ายวิชาการ
    • admin — ผู้ดูแลระบบ
  5. เลือก สังกัด: คณะ / ภาควิชา / หลักสูตร (ถ้ามี)
  6. กดปุ่ม "บันทึกข้อมูล"

✏️ แก้ไขและลบผู้ใช้

  • คลิกที่แถวผู้ใช้ในตารางเพื่อ ขยายรายละเอียด
  • เลือก แก้ไขบทบาท หรือ แก้ไขสังกัด แล้วกด "บันทึก"
  • กดปุ่ม "ลบ" เพื่อลบผู้ใช้ออกจากระบบ
🚫 ไม่สามารถลบบัญชีของตัวเองได้

🏛️ จัดการคณะและภาควิชา

เข้าเมนู Admin → จัดการคณะ

เพิ่มคณะ

  1. กรอก ชื่อคณะ ในช่องด้านบน
  2. กดปุ่ม "บันทึกข้อมูล"

เพิ่มภาควิชา/สาขาวิชา

  1. เลือก คณะ จาก dropdown
  2. กรอก ชื่อภาควิชา
  3. กดปุ่ม "บันทึกข้อมูล"

ดูและลบภาควิชา

ในตารางคณะ กดปุ่ม "ดู/ลบสาขา" เพื่อขยายรายชื่อภาควิชาและลบออกได้

⚠️ ลบภาควิชาได้เฉพาะเมื่อไม่มีหลักสูตรหรือผู้ใช้ที่สังกัดอยู่

📚 จัดการหลักสูตร

เข้าเมนู Admin → จัดการหลักสูตร

➕ เพิ่มหลักสูตรใหม่

  1. เลือก คณะ/ภาควิชา ที่หลักสูตรสังกัด
  2. กรอก ชื่อหลักสูตร
  3. กรอก ปีหลักสูตร (เช่น 2566) — ถ้ามี
  4. กดปุ่ม "บันทึกข้อมูล"

📥 นำเข้ารายวิชาจากไฟล์ (Bulk Import)

สามารถนำเข้ารายวิชาทั้งหมดของหลักสูตรได้ครั้งเดียวผ่านไฟล์

  1. ในตารางหลักสูตร คลิกช่อง อัปโหลดไฟล์ ที่แถวหลักสูตรที่ต้องการ
  2. เลือกไฟล์จากเครื่อง (รองรับ .json, .csv, .txt)
  3. กดปุ่ม "อัปโหลด"

รูปแบบไฟล์ที่รองรับ

รูปแบบหัวคอลัมน์ที่รองรับ
CSV / TSVรหัสวิชา หรือ code, ชื่อวิชา หรือ name, หน่วยกิต หรือ credits
Pipe-delimitedเหมือน CSV แต่คั่นด้วย |
JSONArray ของ object ที่มี field code, name, credits
💡 รองรับหัวคอลัมน์ทั้งภาษาไทยและอังกฤษ — ระบบตรวจจับอัตโนมัติ

📖 จัดการรายวิชา

เข้าเมนู Admin → รายวิชา

➕ เพิ่มรายวิชาด้วยตนเอง

  1. กรอก รหัสวิชา (เช่น CS101)
  2. กรอก ชื่อวิชาภาษาไทย
  3. กรอก ชื่อวิชาภาษาอังกฤษ
  4. กรอก จำนวนหน่วยกิต (เช่น 3(2-2-5))
  5. เลือก หลักสูตร ที่รายวิชาสังกัด
  6. กดปุ่ม "บันทึกข้อมูลวิชา"

🔍 กรอง/ค้นหารายวิชา

ใช้ dropdown "กรองตามหลักสูตร" เพื่อแสดงเฉพาะรายวิชาในหลักสูตรที่ต้องการ

🗑️ ลบรายวิชา

กดปุ่ม "ลบ" ที่แถวรายวิชาในตาราง

⚠️ ไม่ควรลบรายวิชาที่มี Section หรือเอกสาร มคอ. ที่สร้างแล้ว เนื่องจากอาจทำให้ข้อมูลไม่สมบูรณ์

⚙️ ตั้งค่าบัญชี

  1. คลิกรูปโปรไฟล์ที่มุมขวาบน → เลือก "บัญชีของฉัน"
  2. กดปุ่ม "เปลี่ยนรหัสผ่าน"
  3. กรอกรหัสผ่านปัจจุบัน, รหัสผ่านใหม่, และยืนยันรหัสผ่านใหม่
  4. กดปุ่ม "บันทึก"

❓ คำถามที่พบบ่อย

Q: ลืมรหัสผ่านผู้ใช้คนอื่น ทำอย่างไร?

Admin สามารถแก้ไขรหัสผ่านของผู้ใช้ได้จากหน้า จัดการผู้ใช้งาน → คลิกที่ผู้ใช้ → แก้ไขรหัสผ่าน

Q: จะนำเข้าผู้ใช้จำนวนมากได้อย่างไร?

ขณะนี้ยังไม่รองรับการนำเข้าผู้ใช้จำนวนมากจากไฟล์ ต้องสร้างทีละรายการ

Q: ลบหลักสูตรได้หรือไม่?

ได้ แต่ต้องไม่มีรายวิชาหรือ Term Program ที่เชื่อมอยู่ หากมีให้ลบรายวิชาออกก่อน
