admin_panel_settings ภาพรวม Admin

Admin คือผู้ดูแลระบบที่มีสิทธิ์สูงสุด ทำหน้าที่ตั้งค่าโครงสร้างข้อมูลพื้นฐานให้ระบบพร้อมใช้งาน ก่อนที่บทบาทอื่นจะเริ่มดำเนินการได้

หน้าที่รายละเอียด
จัดการผู้ใช้สร้าง/แก้ไข/ลบบัญชีผู้ใช้ทุกบทบาท
จัดการคณะสร้างคณะวิชาและภาควิชา/สาขาวิชา
จัดการหลักสูตรสร้างหลักสูตรและนำเข้ารายวิชาจากไฟล์
จัดการรายวิชาเพิ่ม/ลบรายวิชาในแต่ละหลักสูตร
lightbulb Admin ควรตั้งค่าโครงสร้างข้อมูล (คณะ → หลักสูตร → รายวิชา → ผู้ใช้) ก่อนที่ Academic จะสร้างภาคการศึกษา
groups หลักสูตรและรายวิชา ฝ่ายวิชาการ (Academic) จัดการได้เช่นกัน ส่วนผลการเรียนรู้ PLO/CLO เป็นหน้าที่ของหัวหน้าสาขา (Head) – Admin ไม่ต้องกรอก PLO/CLO เอง
menu_open เมนูฝั่งซ้ายของ Admin: แดชบอร์ด · คณะ / ภาควิชา · หลักสูตร · รายวิชา · ผู้ใช้งาน

login การเข้าสู่ระบบ

  1. เปิดเบราว์เซอร์แล้วไปที่ URL ของระบบ
  2. กรอก ชื่อผู้ใช้ และ รหัสผ่าน แล้วกด "เข้าสู่ระบบ"
  3. ระบบนำไปยัง Admin Dashboard โดยอัตโนมัติ
lightbulb ผู้ใช้ที่มีหลายบทบาทสามารถสลับบทบาทได้ โดยคลิกรูปโปรไฟล์ที่มุมขวาบน แล้วเลือก "สลับบทบาท"

group จัดการผู้ใช้งาน

เข้าเมนู Admin → จัดการผู้ใช้งาน

person_add เพิ่มผู้ใช้ใหม่

  1. กรอก ชื่อผู้ใช้ (Username) – ใช้สำหรับล็อกอิน ไม่สามารถซ้ำกันได้
  2. กรอก รหัสผ่าน – ระบบเข้ารหัสอัตโนมัติ
  3. กรอก ชื่อ-นามสกุล
  4. ติ๊กเลือก บทบาท ที่ต้องการ:
    • instructor – อาจารย์ผู้สอน
    • head – หัวหน้าสาขา
    • academic – ฝ่ายวิชาการ
    • admin – ผู้ดูแลระบบ
  5. เลือก สังกัด: คณะ / ภาควิชา / หลักสูตร (ถ้ามี)
  6. กดปุ่ม "บันทึกข้อมูล"

edit แก้ไขและลบผู้ใช้

  • คลิกที่แถวผู้ใช้ในตารางเพื่อ ขยายรายละเอียด
  • เลือก แก้ไขบทบาท หรือ แก้ไขสังกัด แล้วกด "บันทึก"
  • กดปุ่ม "ลบ" เพื่อลบผู้ใช้ออกจากระบบ
block ไม่สามารถลบบัญชีของตัวเองได้

account_balance จัดการคณะและภาควิชา

เข้าเมนู Admin → จัดการคณะ

เพิ่มคณะ

  1. กรอก ชื่อคณะ ในช่องด้านบน
  2. กดปุ่ม "บันทึกข้อมูล"

เพิ่มภาควิชา/สาขาวิชา

  1. เลือก คณะ จาก dropdown
  2. กรอก ชื่อภาควิชา
  3. กดปุ่ม "บันทึกข้อมูล"

ดูและลบภาควิชา

ในตารางคณะ กดปุ่ม "ดู/ลบสาขา" เพื่อขยายรายชื่อภาควิชาและลบออกได้

warning ลบภาควิชาได้เฉพาะเมื่อไม่มีหลักสูตรหรือผู้ใช้ที่สังกัดอยู่

menu_book จัดการหลักสูตร

เข้าเมนู Admin → จัดการหลักสูตร

add_circle เพิ่มหลักสูตรใหม่

  1. เลือก คณะ/ภาควิชา ที่หลักสูตรสังกัด
  2. กรอก ชื่อหลักสูตร
  3. กรอก ปีหลักสูตร (เช่น 2566) – ถ้ามี
  4. กดปุ่ม "บันทึกข้อมูล"

upload_file นำเข้ารายวิชาจากไฟล์ (Bulk Import)

แทนที่จะเพิ่มรายวิชาทีละวิชา สามารถนำเข้ารายวิชาทั้งหมดของหลักสูตรพร้อมกันได้จากไฟล์เดียว

  1. ในตาราง หลักสูตร หาแถวของหลักสูตรที่ต้องการ แล้วคลิกช่อง อัปโหลดไฟล์
  2. เลือกไฟล์จากเครื่อง – รองรับ .json, .csv, .txt
  3. กดปุ่ม "อัปโหลด" – ระบบจะตรวจสอบไฟล์แล้วนำเข้าให้ทันที
warning ไฟล์ต้องมีขนาดไม่เกิน 2 MB และบันทึกเป็น UTF-8 (เพื่อให้ภาษาไทยไม่เพี้ยน)

คอลัมน์/ฟิลด์ของข้อมูลรายวิชา

ฟิลด์จำเป็น?ชื่อหัวคอลัมน์ที่ระบบรู้จัก (ไทย/อังกฤษ)
รหัสวิชาจำเป็นรหัสวิชา, รหัส, code
ชื่อวิชา (ไทย)จำเป็นชื่อวิชา, ชื่อรายวิชา, ชื่อวิชาไทย, name
ชื่อวิชา (อังกฤษ)ไม่บังคับชื่ออังกฤษ, ชื่อวิชาภาษาอังกฤษ, name_en
หน่วยกิตไม่บังคับหน่วยกิต, credits (เช่น 3(2-2-5))
คำอธิบายรายวิชาไม่บังคับคำอธิบาย, คำอธิบายรายวิชา, description
lightbulb ระบบตรวจจับหัวคอลัมน์ทั้งภาษาไทยและอังกฤษให้อัตโนมัติ ไม่จำเป็นต้องเรียงคอลัมน์ตามลำดับ – ยกเว้นไฟล์ .txt ที่ไม่มีหัวคอลัมน์ ให้เรียงตามลำดับนี้: รหัสวิชา | ชื่อไทย | ชื่ออังกฤษ | คำอธิบาย | หน่วยกิต

ตัวอย่างรูปแบบไฟล์ (ของจริง)

1) CSV .csv – ต้องมีบรรทัดหัวคอลัมน์เสมอ คั่นด้วยจุลภาค (,) สลับลำดับคอลัมน์ได้

รหัสวิชา,ชื่อวิชา,ชื่ออังกฤษ,หน่วยกิต,คำอธิบาย
CS101,การเขียนโปรแกรม 1,Programming I,3(2-2-5),พื้นฐานการเขียนโปรแกรม
CS102,โครงสร้างข้อมูล,Data Structures,3(2-2-5),

2) TXT แบบมีหัวคอลัมน์ .txt – คั่นด้วย | (หรือ Tab / ; ) สลับลำดับคอลัมน์ได้

รหัสวิชา | ชื่อวิชา | หน่วยกิต
CS101 | การเขียนโปรแกรม 1 | 3(2-2-5)
CS102 | โครงสร้างข้อมูล | 3(2-2-5)

3) TXT แบบไม่มีหัวคอลัมน์ – ต้องเรียงตามลำดับนี้เท่านั้น ช่องที่ไม่มีข้อมูลให้เว้นว่าง (เช่น ไม่มีชื่ออังกฤษ/คำอธิบาย)

ลำดับ: รหัสวิชา | ชื่อไทย | ชื่ออังกฤษ | คำอธิบาย | หน่วยกิต

CS101 | การเขียนโปรแกรม 1 | Programming I | พื้นฐานการเขียนโปรแกรม | 3(2-2-5)
CS102 | โครงสร้างข้อมูล | | | 3(2-2-5)

4) JSON .json – เป็น Array ของ object (ชื่อฟิลด์ใช้ไทยหรืออังกฤษก็ได้)

[
  { "code": "CS101", "name": "การเขียนโปรแกรม 1",
    "name_en": "Programming I", "credits": "3(2-2-5)",
    "description": "พื้นฐานการเขียนโปรแกรม" },
  { "code": "CS102", "name": "โครงสร้างข้อมูล", "credits": "3(2-2-5)" }
]
data_object JSON ยังรองรับรูปแบบ จับคู่ตามรหัสวิชา ด้วย เช่น { "courses": { "CS101": { "name": "..." }, ... } }
rule กฎสำคัญ: ทุกแถว/ทุก object ต้องมี รหัสวิชา และ ชื่อวิชาไทย – ถ้ามีแม้แต่แถวเดียวที่ขาด ระบบจะ ปฏิเสธทั้งไฟล์ (ไม่นำเข้าบางส่วน) และแจ้งเลขบรรทัด/รายการที่ผิด
sync นำเข้าซ้ำได้อย่างปลอดภัย: ระบบจับคู่รายวิชาด้วย รหัสวิชา ภายในหลักสูตรเดียวกัน – ถ้ารหัสซ้ำกับที่มีอยู่จะ อัปเดต ข้อมูลเดิม ถ้าเป็นรหัสใหม่จะ เพิ่มใหม่ (ไม่เกิดข้อมูลซ้ำ) หลังอัปโหลดจะแจ้งผลว่า เพิ่มใหม่ / อัปเดต / ข้าม กี่รายการ
error ถ้าไฟล์มีปัญหา (เช่น แถวที่ขาด รหัสวิชา หรือ ชื่อวิชาไทย) ระบบจะแสดง "ตรวจสอบไฟล์ไม่ผ่าน" พร้อมเลขบรรทัดที่ผิด และจะไม่นำเข้าจนกว่าจะแก้ไฟล์ให้ถูกต้อง

book_2 จัดการรายวิชา

เข้าเมนู Admin → รายวิชา

add_circle เพิ่มรายวิชาด้วยตนเอง

  1. กรอก รหัสวิชา (เช่น CS101)
  2. กรอก ชื่อวิชาภาษาไทย
  3. กรอก ชื่อวิชาภาษาอังกฤษ
  4. กรอก จำนวนหน่วยกิต (เช่น 3(2-2-5))
  5. เลือก หลักสูตร ที่รายวิชาสังกัด
  6. กดปุ่ม "บันทึกข้อมูลวิชา"

search กรอง/ค้นหารายวิชา

ใช้ dropdown "กรองตามหลักสูตร" เพื่อแสดงเฉพาะรายวิชาในหลักสูตรที่ต้องการ

delete ลบรายวิชา

กดปุ่ม "ลบ" ที่แถวรายวิชาในตาราง

warning ไม่ควรลบรายวิชาที่มี Section หรือเอกสาร มคอ. ที่สร้างแล้ว เนื่องจากอาจทำให้ข้อมูลไม่สมบูรณ์

settings ตั้งค่าบัญชี

เข้าหน้านี้โดยคลิกรูปโปรไฟล์มุมขวาบน → เลือก "บัญชีของฉัน" หน้าจัดการบัญชีมี 3 ส่วน

badge ข้อมูลบัญชี

กดปุ่ม "แก้ไข" ที่การ์ด ข้อมูลบัญชี เพื่อปรับ ชื่อผู้ใช้, ชื่อ-นามสกุล, โทรศัพท์ และ ห้องพัก

draw ลายเซ็น

ที่การ์ด ลายเซ็น กดปุ่ม "เพิ่มลายเซ็น" (หรือ "แก้ไขลายเซ็น" ถ้ามีอยู่แล้ว) จะเปิดหน้าต่างที่มี 2 วิธีให้เลือก:

วิธีรายละเอียด
วาดลายเซ็นวาดด้วยเมาส์หรือนิ้วในกรอบที่กำหนด – กด "ล้าง" เพื่อเริ่มใหม่
อัปโหลดไฟล์เลือกไฟล์ภาพ PNG, JPG หรือ WEBP ขนาดไม่เกิน 750 KB
  1. เลือกแท็บที่ต้องการ แล้ววาดหรือเลือกไฟล์
  2. กดปุ่ม "บันทึกลายเซ็น" – ระบบจะแสดงตัวอย่างลายเซ็นบนการ์ด
  3. ต้องการลบ ให้กดปุ่ม "ลบ" ที่การ์ดลายเซ็น (มีการยืนยันก่อน)
lightbulb ลายเซ็นใช้สำหรับช่องลงนามในเอกสาร มคอ. ที่พิมพ์/ส่งออก (สำหรับผู้ที่ต้องลงนามในเอกสาร)

lock เปลี่ยนรหัสผ่าน

  1. กดปุ่ม "เปลี่ยนรหัสผ่าน" ที่การ์ด ความปลอดภัย
  2. กรอก รหัสผ่านปัจจุบัน, รหัสผ่านใหม่, และ ยืนยันรหัสผ่านใหม่
  3. กดปุ่ม "บันทึก"
warning รหัสผ่านใหม่ต้องมีอย่างน้อย 6 ตัวอักษร และทั้งสองช่องต้องตรงกัน

help คำถามที่พบบ่อย

Q: ลืมรหัสผ่านผู้ใช้คนอื่น ทำอย่างไร?

Admin สามารถแก้ไขรหัสผ่านของผู้ใช้ได้จากหน้า จัดการผู้ใช้งาน → คลิกที่ผู้ใช้ → แก้ไขรหัสผ่าน

Q: จะนำเข้าผู้ใช้จำนวนมากได้อย่างไร?

ขณะนี้ยังไม่รองรับการนำเข้าผู้ใช้จำนวนมากจากไฟล์ ต้องสร้างทีละรายการ

Q: ลบหลักสูตรได้หรือไม่?

ได้ แต่ต้องไม่มีรายวิชาหรือ Term Program ที่เชื่อมอยู่ หากมีให้ลบรายวิชาออกก่อน
