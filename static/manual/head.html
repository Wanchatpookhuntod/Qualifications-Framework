school ภาพรวม Head

หัวหน้าสาขาเป็นผู้ควบคุมคุณภาพหลักสูตรและเอกสาร มคอ. ในสาขาวิชาที่รับผิดชอบ

หน้าที่รายละเอียด
กำหนด PLOsสร้าง/แก้ไขผลการเรียนรู้ระดับหลักสูตร (Program Learning Outcomes)
กำหนด CLOsสร้าง/แก้ไขผลการเรียนรู้ระดับรายวิชา และเชื่อมโยงกับ PLO
ดูความสัมพันธ์ PLO–CLOตรวจ Matrix ว่าทุก PLO ถูกครอบคลุมด้วย CLO ของรายวิชา
มอบหมายอาจารย์กำหนดอาจารย์ผู้สอนให้กับแต่ละ Section
ตรวจสอบและอนุมัติรีวิว มคอ.3 / มคอ.4 และ มคอ.5 แล้วอนุมัติหรือตีกลับ
จัดทำรายงานสรุปสรุปภาพรวม มคอ.5 ของทั้งสาขาในแต่ละภาคการศึกษา

dashboard Dashboard

หน้า Dashboard แสดงรายวิชาทั้งหมดในสาขา จัดกลุ่มตามภาคการศึกษา

  • เห็น สถานะ มคอ.3 และ มคอ.5 ของแต่ละ Section (DRAFT, SUBMITTED, RETURNED, APPROVED)
  • เห็น ชื่ออาจารย์ ที่ได้รับมอบหมาย (หรือ "รอการมอบหมาย")
  • มีปุ่มลิงก์ไปยังหน้าตรวจสอบเมื่อเอกสารมีสถานะ SUBMITTED
  • ข้อมูลอัปเดตทุก 30 วินาที
lightbulb ใช้ dropdown ภาคการศึกษา ด้านบนเพื่อดูข้อมูลของแต่ละภาค

flag ผลการเรียนรู้ PLO / CLO

หัวหน้าสาขากำหนดผลการเรียนรู้ 2 ระดับ ซึ่งเป็นฐานให้อาจารย์เชื่อมโยงใน มคอ.3

ระดับความหมายกำหนดที่ไหน
PLOProgram Learning Outcomes – ผลการเรียนรู้ของหลักสูตรเมนู PLOs หลักสูตร
CLOCourse Learning Outcomes – ผลการเรียนรู้ของรายวิชา (เชื่อมไปยัง PLO)เมนู CLOs รายวิชา

flag จัดการ PLOs หลักสูตร

เข้าเมนู PLOs หลักสูตร

  1. เลือก หลักสูตร ที่ต้องการจาก dropdown (เห็นเฉพาะหลักสูตรในความรับผิดชอบของคุณ)
  2. กรอก หมายเลข PLO (ตัวเลข เช่น 1, 2, 3 – ระบบจะตั้งรหัสเป็น PLO1, PLO2…) และ คำอธิบาย
  3. กดปุ่ม "เพิ่ม PLO"
  4. แก้ไขหรือลบ PLO แต่ละข้อได้จากแถวในตาราง

checklist จัดการ CLOs รายวิชา

เข้าเมนู CLOs รายวิชา

  1. เลือก หลักสูตร และ รายวิชา ที่ต้องการ
  2. กรอก รหัส/คำอธิบาย CLO ของรายวิชานั้น
  3. ติ๊กเลือก PLO ที่ CLO ข้อนั้นเชื่อมโยงถึง
  4. กดบันทึก – แก้ไข/ลบ CLO ได้จากแถวในตาราง
lightbulb CLO ที่กำหนดไว้จะถูกดึงไปแสดงเป็นค่าตั้งต้นให้อาจารย์เมื่อกรอก มคอ.3 ของรายวิชานั้น

hub ความสัมพันธ์ PLO–CLO

เข้าเมนู ความสัมพันธ์ PLO–CLO เพื่อดู Matrix ภาพรวมของหลักสูตร

  • แสดงว่าแต่ละ PLO ถูกครอบคลุมด้วย CLO ของรายวิชาใดบ้าง
  • ช่วยตรวจหา PLO ที่ ยังไม่มี CLO รองรับ (ช่องว่างของหลักสูตร)
warning หาก PLO ใดไม่ถูกเชื่อมโยงเลย ควรเพิ่ม/ปรับ CLO ของรายวิชาให้ครอบคลุม เพื่อให้หลักสูตรสมบูรณ์

book_2 ดูรายวิชา

เข้าเมนู รายวิชา เพื่อดูรายวิชาทั้งหมดในหลักสูตรที่ดูแล พร้อม CLO ของแต่ละวิชา กรองตามหลักสูตรหรือภาคการศึกษาได้

lightbulb หน้านี้สำหรับ ดู เป็นหลัก การเพิ่ม/ลบรายวิชาเป็นสิทธิ์ของ Admin และฝ่ายวิชาการ

assignment_ind มอบหมายอาจารย์ผู้สอน

ทำหลังจาก Academic เปิดรายวิชา (สร้าง Section) แล้ว – ใช้เมนู มอบหมายผู้สอน หรือทำจาก Dashboard ก็ได้

  1. เข้าเมนู มอบหมายผู้สอน แล้วเลือก ภาคการศึกษา จาก dropdown
  2. ในตารางรายวิชา ดูคอลัมน์ "อาจารย์"
  3. เลือก ชื่ออาจารย์ จาก dropdown ในแถวที่ต้องการ
  4. กดปุ่ม "บันทึก" – อาจารย์จะเห็น Section นี้บน Dashboard ของตน
warning Section ที่ยังไม่ได้มอบหมายอาจารย์ จะแสดง "-- รอการมอบหมาย --" อาจารย์จะไม่สามารถเห็นหรือกรอก มคอ. ได้จนกว่าจะมอบหมาย

fact_check ตรวจสอบและอนุมัติเอกสาร มคอ.

เมื่ออาจารย์ส่งเอกสาร สถานะจะเปลี่ยนเป็น SUBMITTED และปุ่มตรวจสอบจะปรากฏ

description ตรวจสอบ มคอ.3

  1. บน Dashboard กดปุ่ม "ตรวจสอบ" ที่แถวของ มคอ.3 ที่มีสถานะ SUBMITTED
  2. หน้า Review จะเปิดขึ้น แสดง:
    • ข้อมูลรายวิชา (รหัส, ชื่อ, ชื่ออาจารย์, หมู่เรียน)
    • จำนวน CLO ที่กำหนด
    • Preview ข้อมูลสำคัญ
    • เนื้อหาเอกสาร มคอ.3 แบบอ่านอย่างเดียว
  3. ตรวจสอบความถูกต้องและครบถ้วน
  4. กรอก ความคิดเห็น/หมายเหตุ (บังคับกรอก)
  5. เลือกการดำเนินการ (ดูหัวข้อถัดไป)

bar_chart ตรวจสอบ มคอ.5

  1. บน Dashboard กดปุ่ม "ตรวจสอบ" ที่แถวของ มคอ.5 ที่มีสถานะ SUBMITTED
  2. หน้า Review จะแสดง:
    • ข้อมูลรายวิชาและอาจารย์
    • สถิตินักศึกษา (ลงทะเบียน/สำเร็จ/ถอน)
    • การกระจายเกรด
    • ปัญหาที่พบและการปรับปรุง
    • เนื้อหาเอกสาร มคอ.5 แบบอ่านอย่างเดียว
  3. กรอก ความคิดเห็น แล้วเลือกการดำเนินการ

rule อนุมัติ vs ตีกลับ

การดำเนินการผลเมื่อไหร่ควรใช้
check_circle อนุมัติและรับรอง สถานะเปลี่ยนเป็น APPROVED
เอกสารล็อค แก้ไขไม่ได้		 เอกสารถูกต้อง ครบถ้วน
undo ตีกลับเพื่อให้แก้ไข สถานะเปลี่ยนเป็น RETURNED
อาจารย์รับความคิดเห็นและแก้ไขใหม่		 เอกสารมีข้อผิดพลาด หรือไม่ครบถ้วน
warning ต้องกรอก ความคิดเห็น ทุกครั้ง ทั้งเมื่ออนุมัติและตีกลับ เพื่อเป็นหลักฐานการตรวจสอบ
block เมื่ออนุมัติ มคอ.3 แล้ว ระบบจะเปิดให้อาจารย์ส่ง มคอ.5 ได้ – ตรวจสอบให้ครบถ้วนก่อนอนุมัติ

summarize รายงานสรุป มคอ.5 ระดับสาขา

เข้าเมนู Head → สรุป มคอ.5 ของภาคการศึกษาที่ต้องการ

  1. ระบบรวบรวมข้อมูลจากทุก มคอ.5 ที่ APPROVED ในภาคการศึกษา
  2. กรอก ความเห็นภาพรวมระดับสาขา
  3. กรอก ผลการตรวจสอบ/รับรองระดับสาขา
  4. กดปุ่ม "บันทึก/ส่งรายงานสรุป"
lightbulb รายงานสรุปนี้เป็นเอกสารระดับสาขา ไม่ใช่ระดับรายวิชา ควรทำหลังจาก มคอ.5 ทุกฉบับได้รับการอนุมัติแล้ว

folder ดูเอกสารทั้งหมดในภาคการศึกษา

เข้าเมนู เอกสารประจำภาค ในแถวภาคการศึกษา

  • ดูรายการ มคอ.3 และ มคอ.5 ของทุกรายวิชาในสาขา
  • เห็นสถานะและวันที่ส่ง/อนุมัติ
  • คลิกเพื่อเปิดดูเอกสารฉบับเต็ม

settings ตั้งค่าบัญชี

เข้าหน้านี้โดยคลิกรูปโปรไฟล์มุมขวาบน → เลือก "บัญชีของฉัน" หน้าจัดการบัญชีมี 3 ส่วน

badge ข้อมูลบัญชี

กดปุ่ม "แก้ไข" ที่การ์ด ข้อมูลบัญชี เพื่อปรับ ชื่อผู้ใช้, ชื่อ-นามสกุล, โทรศัพท์ และ ห้องพัก

draw ลายเซ็น

ที่การ์ด ลายเซ็น กดปุ่ม "เพิ่มลายเซ็น" (หรือ "แก้ไขลายเซ็น" ถ้ามีอยู่แล้ว) จะเปิดหน้าต่างที่มี 2 วิธีให้เลือก:

วิธีรายละเอียด
วาดลายเซ็นวาดด้วยเมาส์หรือนิ้วในกรอบที่กำหนด – กด "ล้าง" เพื่อเริ่มใหม่
อัปโหลดไฟล์เลือกไฟล์ภาพ PNG, JPG หรือ WEBP ขนาดไม่เกิน 750 KB
  1. เลือกแท็บที่ต้องการ แล้ววาดหรือเลือกไฟล์
  2. กดปุ่ม "บันทึกลายเซ็น" – ระบบจะแสดงตัวอย่างลายเซ็นบนการ์ด
  3. ต้องการลบ ให้กดปุ่ม "ลบ" ที่การ์ดลายเซ็น (มีการยืนยันก่อน)
lightbulb ลายเซ็นจะปรากฏในช่องลงนามของเอกสาร มคอ. ที่พิมพ์/ส่งออกเป็น Word หรือ PDF (เช่น ช่อง "ประธานบริหารหลักสูตร" หากท่านเป็นผู้ลงนาม)

lock เปลี่ยนรหัสผ่าน

  1. กดปุ่ม "เปลี่ยนรหัสผ่าน" ที่การ์ด ความปลอดภัย
  2. กรอก รหัสผ่านปัจจุบัน, รหัสผ่านใหม่, และ ยืนยันรหัสผ่านใหม่
  3. กดปุ่ม "บันทึก"
warning รหัสผ่านใหม่ต้องมีอย่างน้อย 6 ตัวอักษร และทั้งสองช่องต้องตรงกัน

help คำถามที่พบบ่อย

Q: อนุมัติ มคอ.3 ไปแล้ว แต่อยากให้อาจารย์แก้ไขเพิ่มเติม ทำได้ไหม?

ไม่ได้โดยตรง เอกสารที่ APPROVED จะล็อค หากต้องการให้แก้ไขต้องติดต่อ Admin หรือ Academic เพื่อดำเนินการในระดับระบบ

Q: มีอาจารย์ส่ง มคอ.3 แล้วแต่ Head ไม่ได้รับแจ้ง ทำอย่างไร?

ตรวจสอบ Dashboard โดยกรองตามภาคการศึกษา Section ที่สถานะ SUBMITTED จะมีสีเหลือง กดปุ่ม "ตรวจสอบ" ได้เลย

Q: ตีกลับ มคอ.3 ไปแล้ว อาจารย์จะเห็นความคิดเห็นที่กรอกไว้ไหม?

ใช่ อาจารย์จะเห็นความคิดเห็นของ Head ที่ด้านบนของฟอร์ม มคอ.3 เมื่อเปิดหน้าแก้ไข

Q: Head คนหนึ่งดูแลได้กี่สาขา?

ขึ้นอยู่กับการกำหนดสังกัดโดย Admin ถ้า Admin กำหนดสังกัดหลายสาขา Head จะเห็นรายวิชาของทุกสาขาที่สังกัด
