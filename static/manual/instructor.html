edit_document ภาพรวม Instructor

อาจารย์ผู้สอนมีหน้าที่จัดทำเอกสาร มคอ. 2 ประเภท สำหรับแต่ละรายวิชาที่รับผิดชอบ

เอกสารชื่อเต็มจัดทำเมื่อไหร่
มคอ.3 รายละเอียดของรายวิชา (Course Specification) ต้นภาคการศึกษา — ก่อนหรือเมื่อเริ่มสอน
มคอ.5 รายงานผลการดำเนินการของรายวิชา (Course Report) หลังสิ้นสุดภาคการศึกษา — หลังประกาศผลแล้ว
warning ต้องได้รับ มคอ.3 ที่ APPROVED ก่อน จึงจะส่ง มคอ.5 ได้ (บันทึกฉบับร่างได้ก่อน)

dashboard Dashboard

หน้า Dashboard แสดงรายวิชาทั้งหมดที่คุณได้รับมอบหมาย จัดกลุ่มตามภาคการศึกษา

  • ชื่อรายวิชา, รหัสวิชา, หมู่เรียน, ภาคการศึกษา
  • สถานะ มคอ.3: DRAFT / SUBMITTED / RETURNED / APPROVED
  • สถานะ มคอ.5: DRAFT / SUBMITTED / RETURNED / APPROVED
  • วันที่ส่งล่าสุดของแต่ละเอกสาร
  • ปุ่มลิงก์ไปกรอก มคอ.3 และ มคอ.5
lightbulb ถ้าไม่เห็นรายวิชา แปลว่า Head ยังไม่ได้มอบหมายคุณ ให้ Section นั้น — ติดต่อหัวหน้าสาขา

description มคอ.3 — รายละเอียดรายวิชา

มคอ.3 คืออะไร?

มคอ.3 หรือ รายละเอียดของรายวิชา (Course Specification) คือเอกสารที่อธิบายว่ารายวิชานี้จะสอนอะไร อย่างไร และประเมินอย่างไร จัดทำต้นภาคการศึกษาก่อนสอน

วิธีกรอก มคอ.3

กดปุ่ม "มคอ.3" ที่แถวรายวิชาบน Dashboard

check_circle ระบบจะดึงข้อมูลจาก มคอ.3 ที่เคยอนุมัติแล้วของรายวิชาเดิมมาใส่ให้อัตโนมัติ (Pre-fill) ไม่ต้องกรอกใหม่ทุกครั้ง

หมวดที่ต้องกรอก

#หมวดรายละเอียด
1ข้อมูลรายวิชามหาวิทยาลัย, คณะ, หลักสูตร, รหัสวิชา, ชื่อวิชา (ไทย/อังกฤษ), หน่วยกิต, ชั้นปี, หมู่เรียน, วิชาบังคับก่อน, วันที่ปรับปรุงล่าสุด
2คำอธิบายรายวิชารายละเอียดเนื้อหาวิชาทั้งภาษาไทยและอังกฤษ
3CLO — ผลการเรียนรู้กำหนด CLO (Course Learning Outcomes) พร้อมรหัสและคำอธิบายแต่ละข้อ
4การแมป CLO-PLOเชื่อมโยง CLO แต่ละข้อกับ PLO (Program Learning Outcomes) ของหลักสูตร
5แผนการสอนวิธีการสอน, โครงร่างเนื้อหา, ตารางสัปดาห์การสอน
6แผนการประเมินวิธีการประเมินผล, เกณฑ์คะแนน, น้ำหนักคะแนน

การส่งและติดตามสถานะ

  1. กรอกข้อมูลให้ครบ แล้วกดปุ่ม "บันทึกฉบับร่าง" เพื่อบันทึกระหว่างกรอก
    (สามารถกลับมาแก้ไขได้เรื่อยๆ ขณะที่สถานะยังเป็น DRAFT)
  2. เมื่อพร้อมส่ง กดปุ่ม "ส่งให้หัวหน้าสาขา" — สถานะเปลี่ยนเป็น SUBMITTED
  3. รอหัวหน้าสาขาตรวจสอบ — ติดตามสถานะได้บน Dashboard
  4. เมื่อได้รับ APPROVED เอกสารจะล็อค อ่านได้อย่างเดียว
warning ปุ่ม "ส่งให้หัวหน้าสาขา" จะถูก disabled ถ้าฝ่ายวิชาการยังไม่ได้เปิดรอบ มคอ.3

เมื่อถูกตีกลับ (RETURNED)

  1. เปิดหน้า มคอ.3 ของ Section นั้น
  2. อ่าน ความคิดเห็นของหัวหน้าสาขา ที่แสดงด้านบนฟอร์ม
  3. แก้ไขข้อมูลตามที่ระบุในความคิดเห็น
  4. กดปุ่ม "บันทึกฉบับร่าง" แล้ว "ส่งให้หัวหน้าสาขา" อีกครั้ง
lightbulb สามารถส่งซ้ำได้ไม่จำกัดครั้ง จนกว่าหัวหน้าสาขาจะอนุมัติ

bar_chart มคอ.5 — รายงานผลการดำเนินการ

มคอ.5 คืออะไร?

มคอ.5 หรือ รายงานผลการดำเนินการของรายวิชา (Course Report) คือเอกสารที่รายงานว่า สอนไปจริงอย่างไร นักศึกษาบรรลุผลการเรียนรู้แค่ไหน และมีปัญหา/ข้อเสนอแนะอะไร จัดทำหลังสิ้นสุดภาคการศึกษา

block ต้องมี มคอ.3 ที่ได้รับการ APPROVED ก่อน จึงจะส่ง มคอ.5 ได้
(สามารถบันทึกฉบับร่างไว้ก่อนได้ แต่ยังส่งไม่ได้)

วิธีกรอก มคอ.5

กดปุ่ม "มคอ.5" ที่แถวรายวิชาบน Dashboard

check_circle ข้อมูลรายวิชาและ CLO จะถูกดึงมาจาก มคอ.3 ที่อนุมัติแล้วโดยอัตโนมัติ

หมวดที่ต้องกรอก

#หมวดรายละเอียด
1ข้อมูลรายวิชาดึงจาก มคอ.3 อัตโนมัติ — ตรวจสอบความถูกต้อง
2ชั่วโมงสอนจริงบันทึกชั่วโมงที่สอนจริงเทียบกับแผนใน มคอ.3 และหมายเหตุความแตกต่าง
3สถิตินักศึกษาจำนวนที่ลงทะเบียน, สอบผ่านและสำเร็จ, ถอน/ลาออก
4การกระจายเกรดจำนวนนักศึกษาที่ได้เกรด A, B+, B, C+, C, D+, D, F
5ปัญหาและการปรับปรุงปัญหาที่พบระหว่างการสอนและวิธีแก้ไขที่ดำเนินการแล้ว
6วิธีการตรวจสอบกระบวนการตรวจสอบคุณภาพที่ใช้
7ผลการบรรลุ CLOหลักฐานและข้อมูลที่แสดงว่า CLO แต่ละข้อบรรลุผลหรือไม่
8ข้อเสนอแนะข้อเสนอแนะสำหรับภาคการศึกษาถัดไป (ปรับปรุงการสอน, เนื้อหา, การประเมิน)

การส่งและติดตามสถานะ

  1. ตรวจสอบว่า มคอ.3 ของ Section เดียวกัน ได้รับ APPROVED แล้ว
  2. กรอกข้อมูลทุกหมวดให้ครบ
  3. กดปุ่ม "บันทึกฉบับร่าง" เพื่อบันทึกระหว่างกรอก
  4. กดปุ่ม "ส่งให้หัวหน้าสาขา" — สถานะเปลี่ยนเป็น SUBMITTED
  5. รอหัวหน้าสาขาอนุมัติ
warning ปุ่มส่งจะ disabled ถ้า: (1) มคอ.3 ยังไม่ APPROVED หรือ (2) ฝ่ายวิชาการยังไม่เปิดรอบ มคอ.5

flag คู่มือสถานะเอกสาร

สถานะความหมายสิ่งที่ต้องทำ
DRAFT ฉบับร่าง — กำลังกรอกหรือยังไม่ได้ส่ง กรอกให้ครบแล้วกด "ส่งให้หัวหน้าสาขา"
SUBMITTED ส่งแล้ว — รอหัวหน้าสาขาตรวจสอบ รอผล — ดูสถานะบน Dashboard
RETURNED ตีกลับ — หัวหน้าสาขาต้องการให้แก้ไข อ่านความคิดเห็น → แก้ไข → ส่งใหม่
APPROVED อนุมัติแล้ว — เอกสารสมบูรณ์ ไม่ต้องทำอะไร เอกสารล็อค อ่านอย่างเดียว

settings ตั้งค่าบัญชี

  1. คลิกรูปโปรไฟล์ที่มุมขวาบน → เลือก "บัญชีของฉัน"
  2. กดปุ่ม "เปลี่ยนรหัสผ่าน"

  4. กดปุ่ม "บันทึก"

help คำถามที่พบบ่อย

Q: ทำไมไม่เห็นรายวิชาบน Dashboard?

  • หัวหน้าสาขายังไม่ได้มอบหมายคุณให้กับ Section — ติดต่อหัวหน้าสาขา
  • ฝ่ายวิชาการยังไม่ได้เปิดรายวิชา (สร้าง Section) — ติดต่อฝ่ายวิชาการ

Q: ปุ่มส่ง มคอ. เป็นสีเทา กดไม่ได้?

  • ฝ่ายวิชาการยังไม่ได้เปิดรอบ มคอ.3 หรือ มคอ.5
  • สำหรับ มคอ.5: มคอ.3 ยังไม่ได้รับ APPROVED
  • เอกสารอยู่ในสถานะ SUBMITTED หรือ APPROVED อยู่แล้ว

Q: ส่ง มคอ.3 ไปแล้ว อยากแก้ไข ทำได้ไหม?

ไม่ได้ ขณะที่สถานะ SUBMITTED เอกสารจะล็อค — ต้องรอให้หัวหน้าสาขาตีกลับ (RETURNED) ก่อน จึงจะแก้ไขได้

Q: ถูกตีกลับ ความคิดเห็นของหัวหน้าอยู่ที่ไหน?

เมื่อสถานะเป็น RETURNED ให้เปิดหน้า มคอ.3 หรือ มคอ.5 ความคิดเห็นจะแสดงอยู่ ด้านบนสุดของฟอร์ม

Q: ลืมรหัสผ่าน ต้องทำอย่างไร?

ติดต่อ Admin ของระบบเพื่อให้รีเซ็ตรหัสผ่านให้

Q: บันทึกฉบับร่างบ่อยแค่ไหนดี?

แนะนำให้กด "บันทึกฉบับร่าง" ทุกครั้งที่กรอกเสร็จแต่ละหมวด เพื่อป้องกันข้อมูลสูญหาย
